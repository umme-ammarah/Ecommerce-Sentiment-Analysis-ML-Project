# E-commerce Product Review Sentiment Analysis (In Progress)

This project aims to classify Amazon/Daraz product reviews into Positive, Negative, and Neutral categories using NLP techniques.

Work in progress:

- Web scraping (BeautifulSoup / Selenium)
- Text preprocessing & tokenisation
- TF-IDF vectorization
- Machine Learning models (Logistic Regression, SVM, Random Forest)
- Accuracy evaluation & confusion matrix

Code will be uploaded shortly.
